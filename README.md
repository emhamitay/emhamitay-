## 🚀 Projects

### [GhostDrop](https://emhamitay.github.io/ghostdrop/) — Drag & Drop Library
A custom React drag-and-drop library built from scratch — no dependencies.  
Features `Draggable` / `Droppable` wrappers, group support, sortable & switchable components, and an optional Ghost Layer for visual drag previews.  
→ [Live Demo](https://emhamitay.github.io/ghostdrop/) · [Source](https://github.com/emhamitay/ghostdrop)
